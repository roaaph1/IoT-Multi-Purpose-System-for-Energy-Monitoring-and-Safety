# IoT Multi-Purpose System for Energy Monitoring and Safety

This project integrates energy consumption monitoring with safety features
such as gas and smoke detection using ESP32 and IoT technology.

## Features
- Energy monitoring and cost calculation
- Gas and smoke detection
- Real-time alerts via Blynk
- Remote control of devices
- Automatic safety actions
